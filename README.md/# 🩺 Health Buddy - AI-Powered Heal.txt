# 🩺 Health Buddy - AI-Powered Healthcare Chatbot

**Health Buddy** is an AI-powered multimodal chatbot built using NLP and Flask that simulates a doctor-patient interaction. It provides basic medical assistance, medicine suggestions, and healthcare guidance in simple, human-like language.

---

## 🚀 Features

- 🤖 Interactive doctor-like chatbot
- 📋 Diagnoses based on user-selected symptoms
- 💊 Recommends medicines and home remedies for common health issues
- 🧠 Uses YAML files to simulate smart Q&A responses
- 🗣️ Voice input via Speech Recognition (optional)
- 🏥 Suggests nearby hospitals for serious cases
- 🔐 User login/signup system (data stored in `users.yml`)

---

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python, Flask
- **Data**: YAML (for chatbot responses & user data)
- **Tools**: Anaconda, Jupyter Notebook, GitHub

## 🧪 How to Run

1. Clone the repository:
  
    git clone https://github.com/Rahulraju-dev/HealthBuddy.git
    cd HealthBuddy
    

2. Create a virtual environment (optional but recommended):
   
    conda create -n healthbuddy-env python=3.10
    conda activate healthbuddy-env
   or
    conda activate chatbot 
        

3. Install dependencies:
    
    pip install flask pyyaml
   

4. Run the Flask app:
    
    python app.py
    

5. Open your browser:
    
    http://127.0.0.1:5000/
